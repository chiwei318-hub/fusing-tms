請建立歷史地點資料庫系統：

資料表：location_history（地點記錄）
  id
  address TEXT              -- 完整地址
  city TEXT                 -- 縣市
  district TEXT             -- 區域
  lat NUMERIC               -- 緯度（geocode後）
  lng NUMERIC               -- 經度
  location_type TEXT        -- pickup / delivery / both
  
  使用統計：
  visit_count INTEGER DEFAULT 1    -- 送達次數
  last_visited_at TIMESTAMPTZ      -- 最後到訪
  first_visited_at TIMESTAMPTZ     -- 第一次到訪
  
  關聯資料：
  customer_ids JSONB        -- 哪些客戶用過
  driver_ids JSONB          -- 哪些司機跑過
  avg_duration_min INTEGER  -- 平均停留時間
  
  商業資訊：
  place_name TEXT           -- 地點名稱（如：蝦皮桃園倉）
  place_type TEXT           -- warehouse/store/residence/office
  notes TEXT                -- 注意事項（如：需要尾門）
  
  created_at TIMESTAMPTZ DEFAULT NOW()
  updated_at TIMESTAMPTZ DEFAULT NOW()
  
  UNIQUE(address)

資料表：customer_addresses（客戶常用地址）
  id
  customer_id INTEGER
  customer_name TEXT
  address TEXT
  label TEXT                -- 「公司」「倉庫」「門市」
  use_count INTEGER DEFAULT 1
  last_used_at TIMESTAMPTZ
  is_favorite BOOLEAN DEFAULT FALSE

-- 從現有 orders 表匯入歷史地點：
INSERT INTO location_history 
  (address, city, district, location_type, 
   visit_count, last_visited_at, first_visited_at)
SELECT 
  pickup_address, pickup_city, pickup_district,
  'pickup',
  COUNT(*) as visit_count,
  MAX(created_at) as last_visited,
  MIN(created_at) as first_visited
FROM orders
WHERE pickup_address IS NOT NULL
GROUP BY pickup_address, pickup_city, pickup_district

UNION ALL

SELECT 
  delivery_address, delivery_city, delivery_district,
  'delivery',
  COUNT(*) as visit_count,
  MAX(created_at) as last_visited,
  MIN(created_at) as first_visited
FROM orders  
WHERE delivery_address IS NOT NULL
GROUP BY delivery_address, delivery_city, delivery_district

ON CONFLICT (address) DO UPDATE SET
  visit_count = location_history.visit_count + EXCLUDED.visit_count,
  last_visited_at = GREATEST(location_history.last_visited_at, EXCLUDED.last_visited_at);

API endpoints：

GET /api/locations/search?q=關鍵字&type=pickup
地址自動完成（輸入框搜尋用）
回傳：[{ address, place_name, visit_count, lat, lng }]

GET /api/locations/frequent?limit=20
最常用地點排行

GET /api/locations/customer/:customerId
特定客戶的常用地址

POST /api/locations/import
批次匯入歷史地點（Excel/CSV）
支援欄位：地址、地點名稱、類型、備註

GET /api/locations/stats
地點統計：
  總地點數
  本月新增地點
  最常用前10名
  各縣市分佈
